# Security Policy

## Supported Versions

This community edition currently supports the latest `0.1.x` release only.

## Reporting a Vulnerability

If you believe you have found a security issue, please report it privately:

- Email: security@rpc-shield.io
- Subject: "rpc-shield security report"

Please include:
- A clear description of the issue
- Steps to reproduce
- Impact assessment
- Any proof‑of‑concept details

We aim to acknowledge reports within **72 hours**, and provide an initial
assessment within **7 days**.

## Disclosure Policy

We follow responsible disclosure:
- Reports are handled privately until a fix is available.
- We will coordinate a public disclosure date with the reporter.

Thank you for helping improve the security of rpc-shield.
